# Stock-Check
주식 조건 검색 및 주가 관리
*신한 Indi 전용

1. 원하는 조건의 주식을 찾기 어려움

2. 신한 알파 어플 사용 시 가격 알람이 늦게 도착해 매매 타이밍을 많이 놓침

3. 근무 시간에 주가를 계속 들여다볼 수 없음

이로 인해 원하는 조건의 주식을 검색 및 주가 실시간 알림이를 개발
